# # Hipotetyczne lokalizacje Atlantydy — interaktywna mapa

Prosta, statyczna strona HTML z mapą Leaflet i markerami najpopularniejszych teorii lokalizacji Atlantydy (Santorini, Azory, Gibraltar, Richat, Bimini, Kuba, Sundaland).

Uruchomienie:
- Wgraj plik `index.html` do repozytorium na GitHub i aktywuj **GitHub Pages** (branch: `main`, folder: `/ (root)`).
- Po kilku minutach strona będzie dostępna pod: `https://<twoj-login>.github.io/<nazwa-repo>/`

Uwaga: mapa używa kafli OpenStreetMap, więc potrzebne połączenie z internetem do pobrania mapy.
